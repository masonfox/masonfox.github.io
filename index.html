    About Me

    Hello, my name is Mason Fox and I am a Front End Web Developer. I have been working in web for 2 years and have a focus in Front End Development: HTML, CSS, and Javascript. I currently work at Service Spring Corp as a Front End Developer and am the founder and lead developer of Squarecard.io.

    Projects

    Squarecard.io
    Squarecard is a webapp focusing on an effective and practical E-business card webapp approach. It is still in alpha, but we, our team, are constantly working on it!
    React-Tube
    A simple React JS interpretation of Youtube. Using Youtube's API, it searches a query using React JS instantly while typing. I created this during a React course on Udemy.
    React Book
    A simple React JS implementation of storing data with Flux. Created during a React JS course on Udemy.
    React Todo
    Basic React JS todo. It uses Firebase.io to sync db changes, so anything you CRUD to the list is live and real time! Created during a React JS course on Udemy!
    FitDesk
    Fitdesk is a timer for standing desk apps. It uses a minimalist design an does exactly what a timer does... times stuff
    Gulp-Starter
    Gulp-Starter is an out of the box gulp build that includes a plethora of modern web development tools. It is intended to excelerate project startup and provide consitency from project to project.
    CCC Website
    Created during my Internship with Christ Community Church (CCC) while attending Bowling Green State University.
    React Imgur-Browser
    A simple React JS interpretation of Imgur. Uses Imgur's api to display categories, images, and single level comments! Created during a React course on Udemy.
